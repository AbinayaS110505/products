# Product Landing Page with Search (Frontend)

## Overview
This project is a **Product Landing Page** built with **React**, inspired by e-commerce platforms like Amazon and Flipkart. It features a responsive UI, a working search bar with autosuggest, and a products grid with images, names, prices, and ratings.  

> Note: This repository includes the **frontend only**. Backend APIs for products and search are assumed to be running separately.

## Features
- **Header with Search Bar**  
  - Type to see autosuggest dropdown
  - Click a suggestion to populate the search bar  

- **Hero Section**  
  - Banner image or promotional content  

- **Products Grid**  
  - Displays multiple products (image, name, price, rating)  
  - Simple hover effect

- **Responsive Design**  
  - Works on Desktop, Tablet, and Mobile  

- **Filters Section** (basic filter like category or price range)  

## Tech Stack
- **Frontend:** React.js, CSS  
- **State Management:** React `useState` & `useEffect`
